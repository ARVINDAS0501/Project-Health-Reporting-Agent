# Project-Health-Reporting-Agent
This AI PMO agent uses pessimistic logic to parse messy tracking data into executive RAG reports.
This project implements an automated Project Management Officer (PMO) agent designed to synthesize multi-layered project tracking datasets into executive-ready insights. The system is engineered around a Pessimistic Cascade Logic, ensuring that if any single quantitative metric track or qualitative communication layer signals a critical failure, the overall status defaults down to protect leadership from hidden operational blind spots.  To handle the realities of uncleaned corporate data, the system relies on structured Data Corruption Fallbacks. Rather than crashing on standard date-delta calculations when encountering #UNPARSEABLE text errors, the extraction pipeline programmatically shifts focus to literal task completion ratios.  By integrating the openai module, the agent utilizes semantic reasoning to parse unstructured stakeholder comments, converting messy human conversations into structured risk insights. The evaluation framework anchors its analysis tightly to the active project timeline window, treating near-term deliverables as high-priority critical-path dependencies. Ultimately, this architecture bridges the gap between rigid tabular tracking metrics and unstructured human communication, delivering deterministic portfolio health statuses alongside concise, plain-English executive summaries.
